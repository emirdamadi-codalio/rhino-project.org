# Analytics

Segment is the default analytics provider as it can connect multiple sources of analytics data to multiple destinations (Google Analytics, Mixpanel, Intercom, etc).

## Configuration

Create a Segment account add a workspace and add a javascript source for the client.

![](/img/segment_source.png)

get the write key for the source and set it as `VITE_SEGMENT_WRITE_KEY`

![](/img/segment_write_key.png)
